# bzl.nvim

A bazel plugin for neovim. Work in progress — currently a walking skeleton.

Goals: browse bazel targets in a [snacks.nvim](https://github.com/folke/snacks.nvim)
picker, filter by runnable/testable, run/build/test targets with live output.

## Requirements

- Neovim >= 0.10
- bazel or bazelisk on your PATH (or set `bazel_cmd`)
